# 🍶 맛 따라, 술 따라
React / Firebase 기반 전통주 검색 사이트


## introduction
다양한 한국의 전통주를 검색할 수 있으며, 특정 지역의 전통주를 필터링 / 그에 어울리는 음식을 추천
